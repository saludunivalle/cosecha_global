# Módulo Compartido

Código compartido entre los módulos web y cosecha.

## Contenido

- `types/` - Tipos TypeScript compartidos
- `utils/` - Utilidades compartidas
- `constants/` - Constantes y mapeos

## Uso

Este módulo debe ser importado por ambos módulos (web y harvest) para mantener consistencia de tipos y evitar duplicación de código.
